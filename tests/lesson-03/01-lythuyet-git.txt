git commit --amend: thêm thay đổi vào commit cuối cùng, ko sửa commit message
git commit --amend -m”<message>”: thêm thay đổi vào commit cuối cùng - update commit message
git restore --staged <file>: chuyen <file> tu vùng staging area về working directory
git reset HEAD~1: đưa 1 commit gần nhất (cuối cùng) từ vùng repository sang working
