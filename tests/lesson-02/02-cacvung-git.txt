Học Automation Test Từ Chưa Biết Gì

a. Tạo 3 file: file1, file2, file3
==> file1, file2, file3 đang ở vùng local

b. Chạy lệnh: git init
==> file1, file2, file3 chuyển sang vùng working directory

c. Chạy lệnh: git commit -m”init project”
==> 3 files vẫn ở vùng working directory

d. Chạy lệnh: git add file1
==>file1 chuyển qua vùng staging area
==>file2, file3: vẫn ở working directory

e. Chạy lệnh: git commit -m”add file1”
==>file1 chuyển qua repository
==>file2, file3: vẫn ở working directory

f. Chạy lệnh: git commit -m”add file”
==>file2, file3: vẫn ở working directory
